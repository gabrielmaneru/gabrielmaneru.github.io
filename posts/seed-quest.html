Seed Quest

High Concept: Seed quest is a 3D dungeon crawler where the main character, a plant pot, is going to defeat enemies while trying to recover the power of the magical seeds.

Instructor: Iker Silvano

Engine: Custom C++ engine from scratch

Language: C++, Custom Blueprint

Overview: Seed Quest is the first 3D game project and 3D engine I worked on during my junior year at Digipen. This was the moment to put in common all the knowledge we gathered to engineer a powerful 3D engine and make a game out of it. We were 13 programmers which resulted in a very challenging coordination and motivation. The game project was ultimately abandoned after the semester, however, the technologies we managed to built were amazing.

Role: Physics Programmer

3D Rigid Body Simulation

I programmed a physics system allowing for primitive collision detection and resolution. I implemented a simple rigid body behavior using Euler integration. To create more complex shapes, I allowed for hierarchical primitives as well as a bounding volume hierarchy for speeding up the broad phase.

Role: Tools Programmer

Dungeon Generator

In the middle of the production, we needed to refactor the game & level design. We previously used large continuous levels that were difficult to create and maintain. Also, any iteration of these levels resulted in several conflicts while merging. My objective was to create a level pipeline that would allow us to create & iterate fast the levels and rebuild them with deterministic designs. We moved from the open world to closed rectangular rooms.

I created a tool that given a tile-map it designs the floor using choosing between floor models, colors and tile properties (such as water, mud or ice). Then I created the environment procedurally by placing props according to some room properties adding a high detail to the scenes. Finally, the resulting props were baked into a single mesh optimizing a lot of its performance. In the end, this tool resulted essential for the game and helped us move from two basic levels into four of them, each having 10-20 connected rooms.
